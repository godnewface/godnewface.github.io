1. Uji-t Berpasangan: Digunakan jika data berasal dari kelompok yang sama (sebelum-sesudah). Data 1 dan Data 2 harus punya jumlah yang sama.

2. Uji-t Independen: Untuk membandingkan dua kelompok berbeda. Panjang data bisa berbeda.

3. Korelasi Pearson: Mengukur hubungan linear antara dua variabel. Data harus berukuran sama.

4. Korelasi Spearman: Uji non-parametrik untuk hubungan antara dua variabel ordinal atau tidak normal.

5. Regresi: Untuk memprediksi nilai variabel dependen berdasarkan variabel independen. Data 1 = Dependen, Data 2 = Independen.

4. Mann-Whitney U: Uji non-parametrik untuk dua kelompok berbeda. Cocok jika data tidak normal.

5. Chi-Square: Masukkan tabel kontingensi (baris = kategori, kolom = frekuensi per kategori). Contoh:

60 40
40 60
20 30

Masukkan tanpa label (seperti 'Pria', 'Wanita', dsb).
